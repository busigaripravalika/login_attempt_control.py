# Login Attempt Control System

## Description

The Login Attempt Control System is a Python application that demonstrates a simple authentication mechanism with account lockout protection. The program limits the number of incorrect login attempts and temporarily locks access after multiple failures, helping prevent unauthorized access and brute-force attacks.

## Features

- Username and password authentication
- Three login attempts
- Temporary account lock after failed attempts
- Easy-to-understand Python code
- Console-based interface

## Requirements

- Python 3.x

## How to Run

1. Clone or download this repository.
2. Open a terminal in the project folder.
3. Run:

```bash
python login_attempt_control.py
```

4. Enter the login credentials.
